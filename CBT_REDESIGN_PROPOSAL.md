## 1. 현재 구조의 핵심 문제 (요약)

| # | 문제 | 근거 |
|---|---|---|

| P6 | 리뷰 필터 변경 시 페이지 강제 리셋 | 보던 위치 잃음 |

## 2. 새 정보구조 (IA) — 사용자 시안 확정안

> 사용자 손그림 시안(`이름 없는 노트북 (19).pdf`) 기준 확정. CBT 메인을 **큰 카드 4개 + 대시보드 스트립** 허브로.

```
/cbt  (메인 허브)
┌──────────────┐  ┌──────────────┐
│ 1. 회차별     │  │ 2. 과목별     │
│  모의고사     │  │  학습         │
└──────────────┘  └──────────────┘
┌──────────────┐  ┌──────────────┐
│ 3. 오답노트   │  │ 4. 나만의     │
│              │  │  시험(신규)   │
└──────────────┘  └──────────────┘
┌─────────────────────────────────┐
│   대시보드 (가로 와이드, 압축)     │
└─────────────────────────────────┘
   │1          │2          │3              │4
 /cbt/exams  /cbt/study  /cbt/wrong-notes  /cbt/custom(신규)
```

진입점:
- **1 회차별** → `/cbt/exams` (회차 모의고사, 60문항·60분 실전)
- **2 과목별** → `/cbt/study` (과목>토픽>세부, 시간제한 없는 연습)
- **3 오답노트** → `/cbt/wrong-notes` (복습)
- **4 나만의 시험** → `/cbt/custom` **신규** — 과목·문항수·제한시간·난이도·빈출도를 직접 골라 맞춤 시험 구성·응시 (기존 "집중 응시" 일반화)
- **대시보드** = 기존 7섹션을 압축한 가로 와이드 요약 (D-Day·정답률·회차·시간 + 취약영역 한 줄)

용어 통일:
- **학습** = 시간제한 없는 자유 연습 (study) / **모의고사** = 실전 시간제한 (exams)
- "응시"는 모의고사·나만의시험에만, "풀기/연습"은 학습에만

## 3. 라우팅 정리 (P2)

| 현재 | 변경안 |
|---|---|
| `/cbt/practice-{subj}-{topic}-{sub}/take` | `/cbt/study/run?subject=&topic=&sub=` |
| `/cbt/practice-{filter}/take` | `/cbt/exams/run?focus={filter}` |
| `/cbt/{examId}/take` | 유지 (`/cbt/{examId}/take`) |

→ ID를 사람이 읽을 수 있는 쿼리로. 한 곳에서 모드(`practice`/`exam`) 분기.

## 4. 화면별 변경 (와이어프레임)

### 4-1. 홈 `/cbt` — 7섹션 → 3존 (P3)

```
┌──────────────────────────────────────────┐
│ [A] D-12  ·  평균 62% · 5회 · 3.2h        │  ← 한 줄 요약 스트립
├──────────────────────────────────────────┤
│ [B]  📌 다음 추천                          │
│   "전기설비 '접지' 정답률 41% — 12문항 연습" │  ← 취약영역 기반 단일 CTA
│            [ 바로 연습 ]                    │
├──────────────────────────────────────────┤
│ [C] ▸ 최근 응시 기록      (접힘)            │
│     ▸ 학습 곡선 / 일일도전 (접힘)           │  ← 아코디언, 기본 접힘
└──────────────────────────────────────────┘
```

### 4-2. 제출 확인 단일화 (P4)

```
현재: [경고 모달] → [최종 확인 모달]   (2번 클릭)
변경: ┌────────────────────────────┐
      │ 미응답 3문항이 있습니다.      │
      │ 그래도 제출할까요?           │
      │   [돌아가기]   [제출]        │   (1번)
      └────────────────────────────┘
미응답 0이면 모달 없이 바로 제출.
```

### 4-3. 리뷰 필터 (P6)

- 필터(전체/정답/오답/체크) 변경 시 **페이지 리셋 폐지**.
- 대안: 단일 스크롤 + 상단 sticky 필터바, "오답만 12개" 식 카운트 노출.

## 5. 단계별 구현 순서 (승인 후)

1. SubTabs 4탭 + 오답노트 노출 (P1) — 작고 효과 큼
2. 용어 통일 (P5)

4. 라우팅 정리 (P2)
